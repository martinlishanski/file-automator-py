# File Organizer Automation

A simple yet powerful Python automation tool that cleans up your messy folders (like Downloads) by categorizing files into specific directories based on their extensions.

## How it works
The script scans a target directory and moves files into:
- **Documents**: .pdf, .docx, .txt
- **Images**: .jpg, .png, .svg
- **Videos**: .mp4, .mov
- **Archives**: .zip, .rar

## Tech Stack
- **Language:** Python 3.x
- **Libraries:** `os`, `shutil` (built-in)

##  Usage
1. Clone the repository.
2. Run `python organizer.py` in your terminal.
